Parameters
input Object
Properties

A mapping of each format ('iife', 'es', etc.) to the path to use for the output.
baseDir String

The path of the project directory.
moduleName String

The name of the module, for the iife output.
shouldBuildProd boolean (Optional)

Whether or not we should also build a production bundle. Defaults to true.
entry String (Optional)

Used to override the default entry value of ${baseDir}/src/index.js.

this.taskHarness(task, projectOrStar, …args) → Promise.<*>

Helper function that runs a task against all projects, or just one of them. It will collect all the results and reject if any of the tasks rejects.

Parameters
task function

The function to run.
projectOrStar String

Either the name of a project, or '*' for all.
args Object (Optional, Repeatable)

Optional additional arguments to pass to task.
Returns Promise.<*>

Resolves with null if all the promises resolve. Otherwise, rejects with a concatenated error.

this.taskPromiseWrapper(promises) → Promise.<*>

Promise.all() rejects immediately as soon as the first Promise rejects. This wrapper will run each Promise to resolution, and if one or more rejected, reject at the end with a concatenated error message.

Parameters
promises Array.<Promise>

The promises to wait on.
Returns Promise.<*>

Resolves with null if all the promises resolve. Otherwise, rejects with a concatenated error.
