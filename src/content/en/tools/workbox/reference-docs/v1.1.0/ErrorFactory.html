ErrorFactory

class

A simple class to make errors and to help with testing.

Constructor

ErrorFactory

new ErrorFactory(errors)

Parameter

errors

Object

A object containing key value pairs where the key is the error name / ID and the value is the error message.

Method

createError

createError(name, thrownError) returns Error

Parameter

name

string

The error name to be generated.

thrownError

Optional

Error

The thrown error that resulted in this message.

Returns

Error 

The generated error.
