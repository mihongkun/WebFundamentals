WorkboxError

This is a class for managing errors thrown by Workbox.

This class will have full fat error messages in dev builds and be slimmed down in prod.

Constructor

new WorkboxError(errorCode, extras)

Construct a new Workbox Error Message.

Parameters
errorCode String

The error code that is associated with the error message and will be used to generate the appropriate error message.
extras Object (Optional)

These are fields that will be passed to the error message to be included if the message wants and exposed under err.extra should the developer need to access the information.
