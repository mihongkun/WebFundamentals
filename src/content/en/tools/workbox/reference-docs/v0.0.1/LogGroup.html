LogGroup

A simple helper to manage the print of a set of logs

Instance Methods

this._closeGroup()

Closes a console group

this._openGroup()

Opens a console group - managing differences in Firefox.

this._printEdgeFriendly()

This prints a simpler log for Edge which has poor group support. For more details see: https://developer.microsoft.com/en-us/microsoft-edge/platform/issues/11363242/

this._printLogDetails(logDetails)

Prints the specific logDetails object.

Parameters
logDetails object

this.addChildGroup(group)

Parameters
group object

this.addLog(logDetails)

Parameters
logDetails object

this.print()

prints out this log group to the console.
