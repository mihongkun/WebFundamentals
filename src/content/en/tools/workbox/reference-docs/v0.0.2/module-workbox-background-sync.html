workbox-background-sync

Queues failed requests and uses the Background Sync API to replay those requests at a later time when the network state has changed.

Install: npm install --save-dev workbox-background-sync

Instance Methods

(static) this.getResponse(id) → Object

This function returns the fetched response for the given id of the request

Parameters
id String

The ID of the request given back by the broaadcast channel
Returns Object

response Fetched response of the request.
